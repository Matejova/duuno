Landing page

Intro graphic - Float animation

Graphic would be broken down into composition of elements,
that has suttle float on it and all of the elements are off set
→ the float effect is not in sync

Current effect in the example- CSS
.floating {
animation-name: floating;
animation-duration: 4s;
animation-iteration-count: infinite;
animation-timing-function: ease-in-out; }

@keyframes floating {
from {
transform: translate(0, 0px); }

65% {
transform: translate(0, 9px);}

to {
transform: translate(0, -0px);}
}
